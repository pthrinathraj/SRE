## ‚úÖ 1. Introduction

Linux is a family of open-source Unix-like operating systems based on the Linux kernel. It is the backbone of modern infrastructure, widely used in servers, containers, networking devices, and embedded systems. Linux is favored in the SRE world for its stability, flexibility, strong CLI tools, and robust security model.

## ‚úÖ 2. Key Components

- **Kernel** ‚Äì Core that manages hardware, processes, memory, and system calls.
- **Shell** ‚Äì Interface between user and kernel (e.g., bash, zsh).
- **File System Hierarchy** ‚Äì `/etc`, `/var`, `/home`, `/usr`, etc.
- **Package Manager** ‚Äì `apt`, `yum`, `dnf`, `zypper`, etc.
- **Init System** ‚Äì `systemd`, `init`, `upstart`.
- **Networking Tools** ‚Äì `ip`, `ss`, `netstat`, `iptables`, etc.
- **Processes & Scheduling** ‚Äì `ps`, `top`, `nice`, `cron`, `at`.
